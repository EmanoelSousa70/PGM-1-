Diferença entre textContent, innerText e innerHTML

textContent: Retorna ou define o conteúdo de texto de um elemento, incluindo os espaços em branco e sem considerar o estilo ou visibilidade. Ele ignora qualquer HTML presente.

// Exemplo de uso:
const elemento = document.getElementById('exemplo');
elemento.textContent = 'Novo texto';

innerText: Retorna ou define o conteúdo de texto visível de um elemento, respeitando o estilo CSS e a visibilidade. Ele também ignora HTML.

// Exemplo de uso:
const elemento = document.getElementById('exemplo');
elemento.innerText = 'Texto visível';

innerHTML: Retorna ou define o conteúdo HTML de um elemento, permitindo a inclusão de tags HTML.

// Exemplo de uso:
const elemento = document.getElementById('exemplo');
elemento.innerHTML = 'Texto em negrito';
